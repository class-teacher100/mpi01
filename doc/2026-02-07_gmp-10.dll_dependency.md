# gmp-10.dll 依存関係調査

作成日: 2026-02-07

## 概要

vcpkgでインストールした `gmp-10.dll` の依存DLLを調査し、実行環境への追加配置が必要かを確認した。

## 調査対象

```
C:\vcpkg\installed\x64-windows\bin\gmp-10.dll
```

## 依存DLL一覧

| DLL | 種別 | 説明 | 追加配置 |
|---|---|---|---|
| `KERNEL32.dll` | OS標準 | Windowsカーネル基本API | 不要 |
| `VCRUNTIME140.dll` | VC++ランタイム | Visual C++ 2015-2022 ランタイム | 不要（VS2022で導入済み） |
| `api-ms-win-crt-convert-l1-1-0.dll` | Universal CRT | 文字変換関数 | 不要（Windows 10以降標準） |
| `api-ms-win-crt-heap-l1-1-0.dll` | Universal CRT | ヒープメモリ管理 | 不要（Windows 10以降標準） |
| `api-ms-win-crt-locale-l1-1-0.dll` | Universal CRT | ロケール処理 | 不要（Windows 10以降標準） |
| `api-ms-win-crt-runtime-l1-1-0.dll` | Universal CRT | CRTランタイム基盤 | 不要（Windows 10以降標準） |
| `api-ms-win-crt-stdio-l1-1-0.dll` | Universal CRT | 標準入出力 | 不要（Windows 10以降標準） |
| `api-ms-win-crt-string-l1-1-0.dll` | Universal CRT | 文字列操作 | 不要（Windows 10以降標準） |

## 結論

すべての依存DLLはWindows 10以降のOS標準またはVisual Studio 2022のインストールにより提供される。
実行環境には `gmp-10.dll` のみを配置すれば動作する。

## 配置手順

```cmd
copy "C:\vcpkg\installed\x64-windows\bin\gmp-10.dll" C:\Users\cre\mpi01\
```

## 調査方法

MSYS2環境の `grep` でバイナリ内の `.dll` 文字列を抽出して特定した。

```bash
grep -aoi '[a-z0-9_]*\.dll' gmp-10.dll | sort -u
grep -aoi 'api-ms[a-z0-9_\-]*\.dll' gmp-10.dll | sort -u
```

Developer Command Prompt が利用可能な場合は `dumpbin` でも確認できる:

```cmd
dumpbin /dependents gmp-10.dll
```
